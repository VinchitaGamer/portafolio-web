Aplicación de Escritorio: Registro de Personal

Sistema de control de asistencia de personal mediante códigos QR dinámicos.

Python Flask SQLite Openpyxl Tkinter

Descripción del Proyecto

He desarrollado una aplicación de escritorio con Python y Flask para la gestión de asistencia de empleados. La aplicación muestra en pantalla un código QR que se actualiza cada 10 segundos. Los empleados pueden escanear este código para registrar su hora de entrada y salida de forma segura y eficiente.

Características Principales

  • Generación de códigos QR dinámicos cada 10 segundos.
  • Registro de asistencia al escanear el QR desde un dispositivo móvil.
  • Almacenamiento de datos en una base de datos SQLite y exportación a Excel.
  • Interfaz gráfica con diseños personalizados para diferentes estados:
  • Registro Exitoso: Confirmación visual para el empleado.
  • Atraso: Notificación si el registro es posterior a la hora de entrada.
  • Token Invalidado: Mensaje de error si el QR ha expirado.

Desafíos Técnicos

El principal reto fue sincronizar la generación del QR en la aplicación de escritorio con la validación en el servidor Flask en un intervalo de 10 segundos para garantizar la seguridad y evitar registros fraudulentos. La integración de la base de datos SQLite con la exportación a Excel también requirió un manejo cuidadoso de los datos para mantener la integridad.
